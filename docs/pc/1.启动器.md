# 启动器相关

## 下载启动器

如果你是windows用户，你可以选择PCL,HCML等启动器。

[下载启动器](https://www.minecraft.net/zh-hans/download)

如果你是linux用户，你可以选择HCML等启动器

[下载启动器](https://www.minecraft.net/zh-hans/download)

如果你是macos用户，你可以选择HCML等启动器。

[下载启动器](https://www.minecraft.net/zh-hans/download)

如果你是手机用户, 请跳转到[手机启动器](/docs/phone/1.启动器.md)

## 账号使用

初次打开启动器 一般会提醒你下载与java有关的文件。

请根据提示进行下载与安装。

以**HCML**启动器为例 如何开始游戏 你也可以点击此处查看**PCl**启动器的使用方法 [点我](/docs/pc/1.启动器pcl.md)

![HCML](/docs/img/pc/PixPin_2025-12-26_21-11-24.png)

点击**左上角**的 没有游戏账户(点击此处添加账户)

![添加账户](/docs/img/pc/PixPin_2025-12-26_21-12-20.png)

点击添加**离线账户(免费)**或**正版账户(需要到官网购买账号)**

离线用户的用户名可以随便填写(**只能使用字母,数字和下划线**)

正版用户请根据启动器提示进行添加

登录完成后 显示应为这样：

> 我这里使用的正版账户,可能和离线稍有出入

![登录成功](/docs/img/pc/PixPin_2025-12-26_21-17-23.png)

## 游戏下载

**前提**

启动器已经添加了账号

开启了**版本隔离**

版本隔离在左侧的**设置**内打开

设置 --> 版本隔离 --> 各版本独立

![版本隔离](/docs/img/pc/PixPin_2025-12-26_21-22-56.png)

**游戏下载**

点击左侧的**下载** 选择你需要的版本

> 请确保你打开了版本隔离和添加了账号

点击版本列表中的版本 可以根据喜好给你下载的版本命名 安装模组加载器 [点我查看模组加载器的详情](/docs/mod.md)

> 玩原版纯净(不包括优化模组,例如钠,小地图mod)无脑选择fabric即可

> 装fabric一定要装fabric api

![下载游戏](/docs/img/pc/PixPin_2025-12-26_21-29-49.png)

下载完成后 点击右下角的**启动游戏** 就能启动游戏了 切换版本可以点击**启动游戏**旁边的**箭头**切换版本
